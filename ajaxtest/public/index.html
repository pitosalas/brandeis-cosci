This is a demo of the AJAX technique.

  • You can try out the "Date" service using theDate
  • To get an AJAX version of this, click on ajaxdemos/jspage
  • This demo relies on four new files in the rails folder
    • app/controllers/ajaxdemos_controller.rb - which defines the jspage and theDate actions
    • app/models/ajaxdemo.rb - which defines a trivial model
    • app/views/ajaxdemos/jspage.html.erb - which returns a webpage containing some AJAX
    • app/views/ajaxdemos/theDate.html.erb - which returns a webpage containing some AJAX
    • config/routes.rb - which connects the URLs /theDate and /ajaxdemos/jspage to the appropriate MVC code
    No other rails procedures were harmed in the production of this ajax demo!
